# BitMoney (BTM)

BitMoney is a next‑generation digital currency designed for real‑world payments.  
Fast settlement, low fees, predictable tokenomics, and a secure Proof‑of‑Stake network.

## Features
- 2‑second block time
- Ultra‑low fees (0.0001 BTM)
- Predictable 1% annual inflation
- Fee burning mechanism
- WASM smart contracts
- Merchant tools and POS integration
- Multi‑chain token support (ERC‑20, SPL, Cosmos)

## Repositories
### bitmoney-core
Native chain implementation (Cosmos‑style).  
Includes consensus, staking, governance, and bank modules.

### bitmoney-contracts
Smart contracts for EVM, Solana, and wrapped tokens.

### bitmoney-wallet
Web, mobile, and desktop wallets.

### bitmoney-explorer
Full block explorer with API backend and frontend UI.

### docs
Whitepaper, roadmap, architecture, and tokenomics documentation.

### branding
Logos, icons, patches, and color palette.
